Keyboard Commands

File Menu

Ctrl + N = New
Ctrl + O = Open
Ctrl + S = Save
Ctrl + A = Save as

Ctrl + E = Export Game Map File
Ctrl + G = Start Game with Map

Ctrl + B = Save As Bitmap

Ctrl + Q = Quit

 

Edit Menu

Ctrl + U = Undo
Ctrl + R = Redo

Ctrl + M = Edit Map Settings
Ctrl + P = Edit Player Settings
Ctrl + V = Victory Conditions

Ctrl + C = Check Map

 

Help Menu

F1 = Online Manual

 

Zoomlevel

F5   = Zoom level 1    =  2 pixels per field
F6    = Zoom level 1.5 =  4 pixels per field
F7    = Zoom level 2   =   8 pixels per field
F8    = Zoom level 3   =  14 pixels per field
F9    = Zoom level 4   =  20 pixels per field
F10 = Zoom level 5   =  28 pixels per field
F11 = Zoom level 6   =  38 pixels per field
F12 = Zoom level 8   =  50 pixels per field

 

In the "View Map in Game" Mode

ESC = Return to Level Editor

 

New Keyboard Command in Game

T = Watch Clock on / off
